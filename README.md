# TMC 24/7 LLC Website

Professional trucking and logistics services website built with Next.js.

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
bun run build
```

## Deploy

This site is deployed on Netlify. See `netlify.toml` for configuration.
